
make ~/.vim directory, then put files or dir in  vimconfig to .vim directory.

other configs must put in ~/.vimrc file.

0. Chinese help

if vim can't show chinese,then add this setting.
	set helplang=cn

if doc files are encoded in utf-8,than set encoding to utf-i
	set encoding=utf-8

coding commands help:
	:help usr_29
	:help usr_30
